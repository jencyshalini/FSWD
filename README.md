# ✍️ AI-Based Signature Forgery Detection using CNN

🚀 A Deep Learning-based Signature Verification System developed to classify **Genuine** and **Forged** handwritten signatures using **CNN**, **InceptionV3**, and **Explainable AI (Grad-CAM)**.

---

## 📌 Project Overview

Signature forgery is a major challenge in areas like banking, legal verification, and identity authentication.  
This project aims to build an intelligent system capable of detecting forged signatures using Deep Learning and Image Processing techniques.

The model is trained using the **GPDS Signature Dataset** and implemented in **Google Colab** using **TensorFlow** and **Keras**.

---

✅ Genuine vs Forged Signature Classification
